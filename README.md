# Viking-Realms-Capstone
Viking Realms is a 2D top-down role-playing game (RPG) developed using the Unity game engine and C#, designed to combine a Viking fantasy setting with magic-based combat, exploration, and progression.

## Project Goals
- Implement a complete RPG gameplay loop
- Focus on combat, exploration, and progression
- Maintain realistic scope for a 15-week semester

## Planned Features
- Player movement and combat
- Enemies and boss encounters
- Inventory and loot system
- Simple dialogue and quests

## Development Status
Currently in planning and design phase.
